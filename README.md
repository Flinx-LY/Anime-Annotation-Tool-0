# Anime-Annotation-Tool

页面的基本框架copy自CVAT

功能模仿 PixelAnnotationTool ，使用分水岭算法分割画面

可以直接加载视频并识别变化较大的帧用于标注（见底部柱形图）

### 使用的js插件：

jquery,jquery.mousewheel,highcharts(图表绘制),DPlayer(视频播放),pako(数据压缩/解压),colpick(调色板)
